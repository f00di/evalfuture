# Copywriting And Brand Prompt

## Purpose

Improve Evalfuture. product copy, service descriptions, disclaimers, and calls to action.

## When To Use It

Use this when updating homepage copy, service cards, contact text, disclaimers, or CTA language.

## Ready-To-Paste Codex Prompt

```text
You are working on Evalfuture., a professional property comparison and financing evaluation website.

Goal: Improve copy while preserving brand, compliance tone, and UI structure.

Requirements:
- Preserve the brand name "Evalfuture." with the trailing period.
- Main CTA should clearly open the comparison flow.
- Use these phrases where appropriate:
  Free Initial Comparison
  Detailed Property Evaluation
  Consulting Session
  Download Your Excel Comparison
- Explain rent-vs-buy, renting out, buying outright, mortgage financing, interest, service charges, rental returns, market movement, and long-term outcomes.
- Do not overpromise returns.
- Do not say this is formal financial advice.
- Include or preserve this disclaimer:
  This comparison is for informational purposes only and does not constitute financial, investment, mortgage, tax, or legal advice.
- Keep contact details:
  M. Kashif Ansari
  Phone: xxxx
  Email: xxxxxx

Inspect current page copy before editing. Keep changes scoped and run available checks.
```

## Acceptance Criteria

- Copy is professional, clear, and financially careful.
- Required CTA and service phrases are present.
- Disclaimer remains visible.
- Build succeeds.
