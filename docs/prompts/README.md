# Evalfuture. Prompt Library

Use these prompts to start future Codex sessions with the right project context and acceptance criteria.

- UI work: `01-ui-polish.md`
- Formula or calculation fixes: `02-calculation-fixes.md`
- XLSX export work: `03-xlsx-export.md`
- GitHub Pages deployment: `04-github-pages-deployment.md`
- Tests and verification: `05-testing.md`
- Lead capture: `06-lead-capture.md`
- Copy and brand updates: `07-copywriting-and-brand.md`

Recommended workflow:

1. Pick the prompt that matches the task.
2. Paste the ready-to-use prompt into Codex.
3. Add any new details for the current change.
4. Ask Codex to inspect the repo before editing.
5. Require build and relevant checks before commit.

Every prompt assumes:
- Preserve the brand name `Evalfuture.` with the trailing period.
- Preserve GitHub Pages compatibility under `/evalfuture/`.
- Keep the calculator dynamic by loan term.
- Do not make unsupported financial-advice claims.
